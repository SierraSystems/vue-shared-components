# Table

The table lets you aggregate a huge amount of data and present it in a clear and orderly way.

## API

| Prop          | Type             | Required | Description                                             | Default |
| ------------- | ---------------- | -------- | ------------------------------------------------------- | ------- |
| heading       | string/object    | false    | value representing heading to display within table      | ""      |
| elements      | array of objects | false    | content of the table                                    | []      |
| styling       | string           | false    | represents the styling you wish to apply for your table | ""      |
| elementStyles | object           | false    | represents row and column styles you wish to apply      | {}      |

### Elements API

The elements array contains objects of the following structure:

| Prop        | Type          | Required | Description                                                  | Default |
| ----------- | ------------- | -------- | ------------------------------------------------------------ | ------- |
| name        | string/object | true     | the key of the key:value pair in the table                   |         |
| value       | string/object | true     | the value of the key:value pair in the table                 |         |
| isValueBold | boolean       | false    | represents if you want your values to be bolded in the table | false   |
| isNameBold  | boolean       | false    | represents if you want your keys to be bolded in the table   | false   |
| isEmptyRow  | boolean       | false    | represents if you want to have an empty row with hr tag      | false   |
